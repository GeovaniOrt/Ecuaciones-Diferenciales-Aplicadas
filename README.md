# Ecuaciones-Diferenciales-Aplicadas

Las ecuaciones diferenciales son herramientas matemáticas que usan derivadas para describir fenómenos reales, como el movimiento, el calor, el crecimiento de poblaciones o la propagación de enfermedades. Estas ecuaciones relacionan una función desconocida con su tasa de cambio (sus derivadas), y se utilizan en campos como la física, ingeniería, biología y economía para predecir cómo funcionan y evolucionan sistemas complejos.

Este repositorio fue creado con el fin de recolectar información, ejercicios y códigos que realice con el fin de entender que son las ecuaciones diferenciales, cómo funcionan, sus aplicaciones notables y en que aspectos cotidianos las encontramos inmersas.
